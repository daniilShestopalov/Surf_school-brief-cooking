# Бизнес-требования (Business Requirements)

| ID | Формулировка | Приоритет | Источник |
| :--- | :--- | :--- | :--- |
| BR-10 | Обеспечение возможности клиентам самостоятельно бронировать места на кулинарные классы через мобильное приложение. | Must have | [domain_model.md](../1-elicitation/domain_model.md), Раздел 3 (Бронирование) |
| BR-20 | Поддержка политики управления отменами для минимизации потерь студии (разделение на раннюю отмену без штрафов и позднюю со сгоранием оплаты). | Must have | [requirements_summary.md](../1-elicitation/requirements_summary.md), Раздел 1 |
| BR-30 | Предоставление клиентам сопутствующей услуги по аренде кухонной экипировки в рамках доступного прокатного фонда. | Must have | [requirements_summary.md](../1-elicitation/requirements_summary.md), Раздел 1 |
| BR-40 | Резервирование места за клиентом на 1 час для совершения ручного перевода средств по реквизитам. | Must have | [requirements_summary.md](../1-elicitation/requirements_summary.md), Раздел 3 |
| BR-50 | Сбор пользовательских оценок шеф-поваров после проведения классов для формирования рейтинга. | Must have | [requirements_summary.md](../1-elicitation/requirements_summary.md), Раздел 2 |
| BR-60 | Своевременное информирование клиентов в случае форс-мажорных отмен со стороны студии, а также напоминание о предстоящих классах. | Must have | [domain_model.md](../1-elicitation/domain_model.md), Раздел 3 (Обработка форс-мажора) |
