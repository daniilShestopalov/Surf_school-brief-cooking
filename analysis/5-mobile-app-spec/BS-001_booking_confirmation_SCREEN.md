## Обзор

Шторка для проверки деталей перед подтверждением записи, выбора опциональной аренды инвентаря и расчета итоговой суммы.

### User Story

> Как пользователь, я хочу видеть итоговую стоимость и иметь возможность арендовать инвентарь,
> чтобы точно знать, за что я плачу при подтверждении брони (US-80, US-90).

## Используемые запросы

### createBooking

**Тип:** REST  
**Метод:** POST  
**Спецификация:** `analysis/api/bookings/api.yaml` → `createBooking`

**Триггер:** Тап на кнопку "Подтвердить запись"

**Параметры:**
| Параметр | Тип | Обязательность | Источник | Описание |
|----------|-----|----------------|----------|----------|
| `Idempotency-Key` | Header | Да | Генерация на клиенте (UUID) | Защита от дублей |
| `slot_id` | UUID | Да | `slot_id` | Слот |
| `seats_count` | Int | Да | Хардкод `1` в MVP | Количество мест |
| `needs_rental_equipment` | Bool | Да | Состояние чекбокса | Аренда |

**Обработка ответа:**
| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Загрузка | — | Спиннер на кнопке CTA |
| Успех | 201 Created | Переход на SCR-005 (закрытие BS) |
| Ошибка 409 | Места заняты | Снекбар "Извините, места на этот класс только что закончились" |
| Ошибка 409 | Инвентарь занят | Снекбар "Инвентарь уже забронирован другим. Продолжить без аренды?" |
| Ошибка 410 | Слот недоступен | Снекбар с ошибкой (класс отменен) |
| Сеть/5xx | — | Снекбар "Ошибка соединения, попробуйте еще раз" |

## Элементы экрана

### 1. Блок аренды инвентаря
| Элемент | Источник данных | Действие |
|---------|-----------------|----------|
| Текст стоимости | `equipment_tariff` | — |
| Чекбокс аренды | Ввод пользователя (true/false) | Динамический пересчет суммы на CTA кнопке |

**Логика:**
- Если `available_equipment_stock == 0` -> чекбокс disabled (неактивен), показывается текст "К сожалению, весь инвентарь на это время забронирован".
- Пересчет суммы: `base_price + (isChecked ? equipment_tariff : 0)`.

### 2. Кнопка подтверждения
| Элемент | Описание |
|---------|----------|
| CTA-кнопка | Текст: "Подтвердить запись ({Сумма} ₽)" |
