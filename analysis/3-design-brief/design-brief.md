# Реестр экранов и интерфейсных единиц (Design Brief)

В данном документе представлен верхнеуровневый реестр всех интерфейсных экранов и модальных окон/шторок (Bottom Sheets), необходимых для реализации мобильного приложения кулинарной студии.

## Основные экраны (Screens)

| Код | Название | Назначение | Трассировка (Требования) |
| :--- | :--- | :--- | :--- |
| **SCR-001** | Экран входа | Ввод номера телефона, согласия на обработку ПД и OTP-кода. | FR-10, FR-20, UC-10, US-10, US-30 |
| **SCR-002** | Экран регистрации | Ввод имени пользователя при первой авторизации. | FR-15, UC-10, US-20 |
| **SCR-003** | Расписание (Главный) | Отображение списка доступных классов с фильтром по датам. | FR-40, FR-45, FR-115, UC-20, UC-70, US-40, US-50, BR-10 |
| **SCR-004** | Детали класса | Подробная информация о выбранном слоте (описание, шеф, адрес). | FR-50, UC-20, US-70 |
| **SCR-005** | Инструкция по оплате | Отображение реквизитов, итоговой суммы и таймера после создания брони. | FR-80, UC-30, US-100, BR-40 |
| **SCR-006** | Профиль и Мои записи | Список активных бронирований пользователя и доступ к настройкам профиля (аллергии). | FR-30, UC-40, UC-50, US-130, FR-85 |
| **SCR-007** | Детали активной брони | Памятка по предстоящему занятию, адрес, статус оплаты и аренды. Точка входа для Push-уведомлений. | FR-110, UC-80, US-150, US-160 |

## Всплывающие элементы (Bottom Sheets / Modals)

| Код | Название | Назначение | Трассировка (Требования) |
| :--- | :--- | :--- | :--- |
| **BS-001** | Оформление бронирования | Расчет стоимости слота и выбор аренды инвентаря перед подтверждением. | FR-60, FR-65, FR-70, FR-75, UC-30, US-80, US-90, BR-30 |
| **BS-002** | Выбор аллергий | Список тегов для отметки аллергий пользователя в профиле. | FR-25, UC-40, US-60 |
| **BS-003** | Отмена бронирования | Подтверждение отмены записи с предупреждением о штрафах (если применимо). | FR-85, FR-90, UC-50, US-110, US-120, BR-20 |
| **BS-004** | Оценка шефа | Шторка для выставления оценки (1-5 звезд) прошедшему классу. | FR-100, FR-105, UC-60, US-140, BR-50 |

*Примечание: Детальное описание каждого экрана представлено в соответствующих файлах в текущей директории.*
